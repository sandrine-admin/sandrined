## 🏗️ Mon Homelab

Un environnement de lab complet tournant sur **Proxmox VE** avec 10 conteneurs LXC et 1 VM :

| ID | Nom | Rôle |
|---|---|---|
| lxc/100 | Pi-hole | DNS filtrant & anti-pub |
| lxc/102 | Uptime Kuma | Monitoring & status page |
| lxc/104 | Nextcloud + Portainer | Cloud privé & gestion Docker |
| lxc/105 | Grafana | Dashboards & alerting |
| lxc/106 | Step CA | PKI interne & certificats TLS |
| lxc/107 | Ollama | Runtime LLM local |
| lxc/108 | Open WebUI | Interface IA locale |
| lxc/109 | Jellyfin | Serveur média |
| lxc/110 | Hermès Agent | Agent IA autonome (Nous Research) |
| qemu/200 | Windows Server 2025 | Active Directory |
